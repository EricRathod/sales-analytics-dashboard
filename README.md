# Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data from the Superstore dataset containing 9,994 records.

The goal was to identify sales trends, profitability, top-performing products, and regional performance using Python data analysis techniques.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

Superstore Sales Dataset

Records: 9,994

Columns: 21

## Analysis Performed

* Sales by Category
* Profit by Category
* Sales by Region
* Top 10 Products by Sales
* Business Insights

## Key Findings

* Technology generated the highest sales.
* Regional performance varied significantly.
* A small number of products generated a large percentage of revenue.
* Profitability differed across product categories.
